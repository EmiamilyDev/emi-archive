# emi-archive

Minimal premium editorial archive site scaffold.

## Pages

- `index.html` - Homepage (hero, quick stats, latest records, quote)
- `archive.html` - Archive list with search, filters, and grid/list toggle
- `record.html` - Archive record detail page
- `timeline.html`, `collections.html`, `about.html` - Supporting pages

## Design Concept Applied

- **Style:** Editorial Archive / Digital Museum / Timeless
- **Palette:** `#F7F4EF`, `#E8E1D7`, `#C9B08A`, `#1C1C1C`
- **Typography:** Playfair Display (headings), Inter (body)

## Run

Open `index.html` directly in a browser, or serve with any static server.
